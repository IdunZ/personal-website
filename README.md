# Personal Website

A collection of static HTML pages serving as a personal website, portfolio, and digital playground. This project is primarily built with **HTML** and is designed to be a simple, fast, and easily deployable web presence.

## ðŸš€ Features

*   **Multi-Page Structure:** Includes dedicated pages for various content types.
*   **Profile Page (`Profile.html`):** A dedicated page for personal information and biography.
*   **Gallery/Portfolio (`Gallery.html`, `Gallery2.html`):** Pages to showcase images, projects, or creative work.
*   **Contact Form (`Contact.html`):** A page for visitors to get in touch.
*   **Article/Blog Section (`Article.html`):** A template for sharing longer-form content or blog posts.
*   **Interactive Elements:** Includes pages for a guest book (`Guest-book.html`) and a simple game section (`game.html`, `game-selengkapnya.html`).
*   **Utility Pages:** Contains pages for user management like login (`login.html`), sign-up (`signup.html`), and editing (`editing.html`, `oprek.html`).

## ðŸ’» Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You only need a modern web browser to view this website.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/IdunZ/personal-website.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd personal-website
    ```
3.  **Open the main file:**
    Simply open the `index.html` file in your preferred web browser.
    ```bash
    # Example command (may vary by OS)
    open index.html
    ```
